# MIPS-like

## Ordem de Leitura
 - 17-13 RT entry 1
 - 18-22 RS entry 2
 - 23-27 RD dest 
 - 28-31 opcode

## OPCODES

  - 0000 : sll
  - 0001 : slr
  - 0010 : MUL
  - 0011 : ADD
  - 0100 : SUB
  - 0101 : AND
  - 0110 : OR
  - 0111 : XOR
  - 1000 : LES
  - 1001 : MulMatrix4x4 
  - 1010 :

  - 1111 : lw
